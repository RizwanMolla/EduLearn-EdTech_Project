# EduLearn Backend

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Set up MongoDB (local or Atlas) and update `.env` if needed.
3. Start the server:
   ```bash
   npm run dev
   ```

## API Endpoints

- `POST /api/auth/signup` — Register a new user
- `POST /api/auth/login` — Login and receive JWT token
